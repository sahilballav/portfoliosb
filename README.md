# Sahil Ballav Portfolio

## Quick Start in VS Code

You have all the files ready. Follow these steps to get the website running:

1.  **Open in VS Code**:
    - Open Visual Studio Code.
    - Go to **File** > **Open Folder...** and select the folder containing these files.

2.  **Install Dependencies**:
    - Go to **Terminal** > **New Terminal** (or press `` Ctrl + ` ``).
    - Type the following command and press Enter:
      ```bash
      npm install
      ```
    - Wait for the installation to finish (it might take a minute).

3.  **Run the App**:
    - In the same terminal, type:
      ```bash
      npm run dev
      ```
    - Press Enter.

4.  **View the Site**:
    - You will see a link like `http://localhost:5173`.
    - Hold **Ctrl** (or Cmd on Mac) and click that link to open it in your browser.

## Configuration

- **API Key**: The API key is already configured in the `.env` file.
- **Edit Content**: Modify `constants.tsx` to update your portfolio data (projects, experience, etc.).
